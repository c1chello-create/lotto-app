행운로또 v1.6.1 - 꿈해몽 AI Preview 테스트 버전

변경 내용
1. 기존 꿈 키워드 → 기본 꿈번호 기능 유지
2. 꿈해몽 AI Preview 체크박스 추가
3. Preview ON 상태에서 기본 꿈번호를 만든 뒤, 최근 50회/100회/전체 설정에 따라 1단계 동반번호 보정 적용
4. 최종 적용번호, 기본 꿈번호, 동반 보정 후보 TOP 표시
5. 꿈의 원래 의미 유지를 위해 기본 꿈번호 4개 이상 우선 보존
6. 동반번호 확장은 1단계에서 종료

업로드 방법
- combo.html
- combo.js
- data/lotto.json
위 구조 그대로 GitHub 저장소에 업로드하세요.

테스트 방법
1. combo.html 실행
2. 꿈해몽 연동 영역에 키워드 입력 예: 도자기, 새, 물
3. 꿈해몽 AI Preview 체크 상태 확인
4. 꿈해몽 번호 적용 버튼 클릭
5. 기본 꿈번호 / 동반 보정 후보 / 최종 적용번호 확인

주의
- 이 버전은 정식 AI Core가 아니라 Preview 테스트입니다.
- 로또 당첨을 보장하지 않으며, 꿈 상징과 과거 회차 데이터의 동반번호를 연결해 보는 분석 기능입니다.
